# 1. DESCRIPCIÓN DEL NEGOCIO

La tienda “GymWear Santander” es un pequeño negocio dedicado a la venta de ropa deportiva para gimnasio, como polos, leggings, shorts y accesorios. Los productos se manejan por tallas (S, M, L), colores y tipos.

El negocio busca mejorar el control de su inventario y ventas mediante un sistema digital.

# 2. IDENTIFICACIÓN DEL PROBLEMA Y SOLUCIÓN

PROBLEMA:

La tienda lleva el control de productos y ventas de forma manual (cuaderno o Excel básico), lo que genera:

* Confusión en el stock disponible
* Pérdida de información
* Errores al registrar ventas
* Dificultad para saber qué productos se venden más

SOLUCIÓN:

Desarrollar un sistema de inventario web que permita:

* Registrar productos con talla y color
* Controlar entradas y salidas de productos
* Visualizar el stock en tiempo real
* Registrar ventas correctamente

# 3. PREANÁLISIS

NECESIDADES

* Control de productos (ropa de gym)
* Registro de ventas
* Control de stock por talla y color
* Acceso rápido a la información

ESTUDIO DE VIABILIDAD

TÉCNICA:
Se utilizará PHP, MySQL y navegador web, tecnologías accesibles y conocidas.

ECONÓMICA:
Bajo costo, ya que se usará software libre.

OPERATIVA:
El sistema será fácil de usar para el dueño o trabajadores sin conocimientos técnicos.

ALCANCE DEL SISTEMA

INCLUYE:

* Registro de productos (nombre, talla, color, precio)
* Registro de ventas
* Control de stock
* Consulta de inventario

NO INCLUYE:

* Pagos en línea
* Facturación electrónica
* Aplicación móvil

# 4. ANÁLISIS

DEFINICIÓN DE REQUISITOS

REQUISITOS FUNCIONALES:

* Registrar productos
* Editar productos
* Eliminar productos
* Registrar ventas
* Mostrar stock disponible
* Buscar productos por nombre, talla o color

REQUISITOS NO FUNCIONALES:

* Interfaz simple y fácil de usar
* Sistema rápido (carga eficiente)
* Seguridad básica (login de usuario)
* Compatible con navegadores web

ANÁLISIS DE REQUISITOS

El sistema debe permitir al usuario gestionar el inventario de forma clara y sin errores, asegurando que la información sea precisa y fácil de consultar.
